- PCC
- RP2040 Zero
- Kailh Hotswap sockets
- MX Style Switches
- Diodes? if your PCB doesnt have them

Sockets: https://www.aliexpress.com/item/1005003704619399.html or other
Switches: https://www.aliexpress.com/item/1005004285423123.html or other
